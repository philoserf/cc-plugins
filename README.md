# cc-plugins

Claude Code plugins by [philoserf](https://github.com/philoserf).

## Installation

```bash
claude plugin marketplace add philoserf/cc-plugins
```

## Plugins

| Plugin                    | Description                                                     |
| ------------------------- | --------------------------------------------------------------- |
| [use-trueup](use-trueup/) | Detects drift between implementation and spec before committing |

Install a plugin:

```bash
claude plugin install use-trueup@cc-plugins
```
